# NFL Model Project

This repository contains code and data for building, training, and evaluating NFL game prediction models.

## Structure

- `data/`: Raw, interim, feature, and output datasets
- `models/`: Trained models and evaluation reports
- `notebooks/`: Data exploration and experiments
- `src/`: Source code for data ingestion, feature engineering, modeling, and pipeline

## Getting Started

1. Create and activate a Python virtual environment
2. Install dependencies: `pip install -r requirements.txt`
3. Run the pipeline: `bash run.sh` (optional)
